# Project Planner

Project part of the Udemy course Build Web Apps with Vue JS 3 & Firebase

## Project setup

```
yarn install
```

### Run json-server

```
npx json-server --watch data/db.json
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```
